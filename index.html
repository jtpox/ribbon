ribbon

A platform to share your stories.

Stable GitHub

Get ribbon

Step 1

Clone the stable branch.

git clone -b stable https://github.com/jtpox/ribbon.git

Step 2

Update database details.

config/database.json

Step 3

Install and start.

npm run setup

node dist/ribbon.js

Install arabesque

A back end interface for ribbon.

Go to the admin folder.

cd admin

Clone arabesque.

git clone https://github.com/jtpox/arabesque.git

Enter the arabesque folder.

cd arabesque

Install dependencies.

npm install

Build for production.

npm run build

Prerequisites

What makes ribbon tick.

NodeJS 8 or newer.

MongoDB 3.6.3 or newer.
